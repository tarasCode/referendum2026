# Referendum 2026 Assistant

Assistente AI per il Referendum 2026, basato su n8n Chat e addestrato con la "Pubblicazione n. 2" del Ministero dell'Interno.

**Live:** [referendum2026.netlify.app](https://referendum2026.netlify.app)

## Stack

- HTML/CSS/JS statico
- [n8n Chat Widget](https://www.npmjs.com/package/@n8n/chat) per l'interfaccia conversazionale
- PWA installabile (Service Worker + Web App Manifest)
- Deploy automatico su Netlify
